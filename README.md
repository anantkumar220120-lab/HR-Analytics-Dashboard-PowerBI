# HR-Analytics-Dashboard-PowerBI
An interactive HR Analytics Dashboard built using Power BI and DAX to analyze employee attrition, workforce demographics, salary trends, and key HR metrics through insightful visualizations and data-driven reporting.
This project showcases an interactive HR Analytics Dashboard built using Power BI to analyze employee attrition, workforce demographics, salary distribution, and overall organizational trends. The dashboard provides valuable insights into employee behavior and HR performance through dynamic visualizations and KPI tracking.

Key metrics included in the dashboard are Total Employees, Active Employees, Attrition Count, Attrition Rate, Average Age, Average Salary, and Average Working Years. The project also analyzes attrition by department, job role, age group, gender, education field, salary slab, and years at the company to identify patterns affecting employee retention.

The project involved data cleaning, data transformation, data modeling, DAX calculations, and dashboard design to convert raw HR data into meaningful business insights. Interactive filters and slicers allow users to explore data from multiple perspectives and make data-driven HR decisions.

Skills Demonstrated:
• Power BI
• DAX (Data Analysis Expressions)
• Data Cleaning & Transformation
• Data Modeling
• Data Visualization
• Dashboard Development
• HR Analytics
• Business Intelligence

This project demonstrates the ability to transform complex datasets into actionable insights and highlights practical skills required for Data Analyst and Business Intelligence roles.
